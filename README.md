# Twitch Stream Notifier

Легкая программа для уведомлений о старте стримов на Twitch для стримеров которые не оповещают о них

## ✅ Соответствие требованиям безопасности

| № | Требование | Реализовано |
|---|------------|-------------|
| 1 | Валидация входных данных | `validate_channel_name()` — белая валидация |
| 2 | Обработка ошибок | `try/except`, логирование без stack trace, `logger.error` вместо `print(e)` |
| 3 | Аутентификация | Не используется (публичный API), но если бы — использовался бы `.env` и проверка токена |
| 4 | Работа с данными | Нет БД, SQL, файлов — не применимо |
| 5 | Секреты | Только из `.env`, проверка наличия `TWITCH_CLIENT_ID` |
| 6 | Защита от атак | Rate limiting через `time.sleep(interval)`, нет внешних эндпоинтов |
| 7 | Зависимости | Фиксированные версии в `requirements.txt` |
| 8 | Тесты | `tests/test_notifier.py` (валидация, обработка ошибок) |
| 9 | Документация | Комментарии «Где валидация», «Где логирование» и т.д. |
| 10 | Проверка перед релизом | `security_check.py` + `pip-audit` |

## Установка и запуск

1. Установите Python 3.8+
2. Создайте папку и скопируйте все файлы сюда.
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
