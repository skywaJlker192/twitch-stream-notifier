# 🎮 Twitch Stream Notifier

[![Python](https://img.shields.io/badge/Python-3.12+-blue.svg)](https://python.org)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Twitch API](https://img.shields.io/badge/Twitch-API-purple.svg)](https://dev.twitch.tv/docs/api)

**Автоматический монитор стримов Twitch с уведомлениями и авто-открытием браузера.**

Приложение отслеживает указанные каналы в реальном времени и мгновенно уведомляет вас, когда стример начинает трансляцию.

---

## ✨ Возможности

- 🔔 **Мгновенные уведомления** — системные уведомления при старте стрима
- 🌐 **Авто-открытие браузера** — стрим открывается автоматически через 1 секунду
- 🔄 **Фоновая работа** — программа работает в фоне после скрытия окна
- 🔐 **Безопасность** — OAuth 2.0 аутентификация, секреты в `.env`
- ✅ **Валидация** — проверка имён каналов по белой списке
- 🎯 **Гибкая настройка** — интервал проверки от 10 до 300 секунд
- 🖼️ **Красивые окна** — превью стрима с информацией о канале

## ⚡ Быстрый старт (для пользователей)

### 1. Скачай exe

Перейди в [Releases](https://github.com/ТВОЙ_НИК/twitch-stream-notifier/releases) и скачай **`TwitchNotifier.exe`** из последнего релиза.

### 2. Получи Twitch API ключи

1. Перейди на [Twitch Developer Console](https://dev.twitch.tv/console/apps)
2. Нажми **"Register Your Application"**
3. Заполни:
   - **Name**: `twitch-notifier` (любое)
   - **OAuth Redirect URL**: `http://localhost:3000`
   - **Category**: `Application Integration`
4. Скопируй **Client ID**
5. Нажми **"New Secret"** → скопируй **Client Secret**

### 3. Получи OAuth токен

Открой в браузере (замени `YOUR_CLIENT_ID`):


```
https://id.twitch.tv/oauth2/authorize?client_id=YOUR_CLIENT_ID&redirect_uri=http://localhost:3000&response_type=token
```

- Авторизуйся в Twitch
- Скопируй токен из адресной строки (после `access_token=`)

### 4. Настройка

Создай файл `.env` в папке проекта:

```env
TWITCH_CLIENT_ID=твои_client_id
TWITCH_OAUTH_TOKEN=твой_oauth_токен
```

### 5. Запуск

```bash
python notifier.py
```

## 📖 Использование

1. **Запуск программы**:
   - Откроется окно настроек
   - Введи `TWITCH_CLIENT_ID` (или он подтянется из `.env`)
   - Добавь каналы для отслеживания (по одному в строке)
   - Укажи интервал проверки (рекомендуется: 30-60 сек)
   - Нажми **"💾 Сохранить и Запустить"**

2. **Работа в фоне**:
   - Окно настроек скроется
   - Программа продолжит работать в фоне
   - Для остановки: найди процесс в Диспетчере задач или нажми `Ctrl+C` в терминале

3. **Когда стрим начнётся**:
   - 📱 Системное уведомление
   - 🖼️ Окно превью с информацией
   - 🌐 Браузер автоматически откроет стрим

## 🐛 Решение проблем

### Ошибка `401 Unauthorized`
- Проверь, что `TWITCH_CLIENT_ID` и `TWITCH_OAUTH_TOKEN` указаны верно
- Токен мог истечь — получи новый

### Ошибка `429 Too Many Requests`
- Увеличь `poll_interval` до 60+ секунд
- Twitch ограничивает количество запросов

### Не приходят уведомления
- Установи `plyer`: `pip install plyer`
- Проверь настройки уведомлений в системе

### Окно не открывается
- Убедись, что установлен `tkinter` (встроен в Python)
- На Windows: `py -m tkinter`
